## Что нужно создать

### Основной endpoint
- **POST /analyze** - принимает видео через multipart/form-data
- Анализирует первые 5-10 секунд с помощью Gemini 2.5 API
- Отправляет результат на наш сервер через callback
- Возвращает JSON с результатами

### Анализируемые элементы
1. Встроенные субтитры
2. Озвучка/голосовое сопровождение
3. Звуковые эффекты или музыка
4. Графические эффекты (VHS, overlay, фильтры)
5. Слайды/текстовые вставки
